# **App Name**: CodeCanvas Portfolio

## Core Features:

- Project Showcase: Display completed projects with descriptions, images, and links to live sites or GitHub repositories.
- Interactive Experience Timeline: Show professional experience chronologically, highlighting roles, responsibilities, and achievements using an interactive timeline.
- Skills Summary: Outline key technical skills and areas of expertise.
- About Me Section: Provide a personal introduction with a photo, brief bio, and contact information.
- Contact Form: A contact form that delivers submissions directly to the engineer's inbox.
- AI-Powered Skill Endorsement Tool: AI Tool that analyzes project descriptions and auto-suggests skills with a confidence level, which can be added to the skills summary to automatically highlight experience and tailor your skill presentation.

## Style Guidelines:

- Dark background (#121212) for a modern, professional look.
- Primary color: Teal (#007997) to convey professionalism and visual interest.
- Accent color: Light teal (#85C6DB) used sparingly for highlights and interactive elements, creating contrast against the dark background.
- Headline font: 'Space Grotesk', sans-serif. Body text: 'Inter', sans-serif.
- Minimalist icons for navigation and representing skills.
- Clean, grid-based layout with clear sections for projects, experience, and about me.
- Subtle animations for transitions and highlighting interactive elements on scroll or hover.
